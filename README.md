# AGN Disk Black Hole Plot

This script makes a simple figure showing black holes inside an AGN disk.

The plot includes:
- A supermassive black hole at the center
- The disk around it
- A migration trap (where objects pile up)
- A pair of black holes merging
- Scattered black holes outside the disk
- New compact objects formed inside the disk

---

## Output

The script saves two files:
- agn_disk.png
- agn_disk.pdf
